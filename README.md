# Speech_Segment_Classification
Voiced, Unvoiced and Silence Speech Classification

## Overview

This project implements speech signal analysis techniques in MATLAB to classify audio segments into voiced speech, unvoiced speech, and silence. The classification process uses framing, energy estimation, pitch detection, autocorrelation, and frequency-domain analysis.

## Objectives

* Segment speech into fixed-length frames.
* Extract speech characteristics using signal processing techniques.
* Identify voiced, unvoiced, and silence regions.
* Visualize and analyze speech behavior.

## Methodology

1. Load and normalize the speech signal.
2. Divide the signal into frames.
3. Calculate frame energy.
4. Estimate pitch using autocorrelation.
5. Classify frames based on energy and pitch thresholds.
6. Visualize classified speech segments.

## Technologies Used

* MATLAB
* Speech Processing
* FFT Analysis
* Autocorrelation
* Pitch Estimation

## Results

* Accurate identification of voiced speech regions.
* Detection of unvoiced speech and silent intervals.
* Visualization of classified speech components.
